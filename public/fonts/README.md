# Fonts

This directory contains the Clash Display font files for the project.

## How to Add Fonts

1. Download **Clash Display** from [Fontshare](https://www.fontshare.com/fonts/clash-display)
2. Extract the downloaded ZIP file
3. Copy the following files to this directory:
   - `ClashDisplay-Variable.woff2` (or individual weight files)

## Required Files

For the variable font approach (recommended):
- `ClashDisplay-Variable.woff2`

Or for individual weights:
- `ClashDisplay-Extralight.woff2` (200)
- `ClashDisplay-Light.woff2` (300)
- `ClashDisplay-Regular.woff2` (400)
- `ClashDisplay-Medium.woff2` (500)
- `ClashDisplay-Semibold.woff2` (600)
- `ClashDisplay-Bold.woff2` (700)

The `globals.css` file is configured to use the variable font by default.
